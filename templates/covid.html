About Covid / Pneumonia Detection Model

The model backbone used here is resnet200d pretrained on imagenet , A bit about why resent200d and why not other architectures such as efficient nets , nfnets , vgg etc . resnet200d is supposed to perform better for xrays images by my experimentations and effnets and other perform not as good as resnet type architecture although i tried all the other architectures that i could possibly try and came to the conclusion that resnet200d indeed is the king for the backbone of my model the final layers are just Linear layers to classify into 3 categories i.e Covid , pneumonia and normal using CrossEntropyloss as loss function with some label smoothing I was thinking to make it as binary classification model by making 3 models for each class but didnt got as good performance as with multiclass model I have used ptyorch framework as i find it more flexible and it suits my working type . A bit about dataset images were already downscaled to 256*256 pixels hence i trained my model with these images and surpsingly model perform way better than i expected , i have done some heavy augmentations using albmunetations library with automated mixed precision training , i tried to use cutmix , mixup , fmix etc augmentations but they gave poor performance . I have also added Grad Cam to see what model is seeing while predicting. I have made the Source code Public it can be found here make sure to give credits if you use it for your research purpose !
